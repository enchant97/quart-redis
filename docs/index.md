# Welcome
An easy way of setting up a redis connection in quart.

## Installation
Install the pip package using pip. The latest compatible release of redis will be automatically installed.

```
pip install quart-redis
```

## Requirements
- quart ~= 0.20
- redis >= 5.2.1, < 6
